# PIPDASHS01_UI.md
## Dashboard UI SSOT

### 1. Layout
- 상단: KPI 카드 영역 (4칸)
- 중단: 자산 유형 비중 차트
- 하단: 지역 노출 비중 차트

### 2. KPI Cards
1. 총 자산
2. 총 투자 원금
3. 평가 손익
4. 총 수익률

### 3. Charts
- Asset Allocation: Pie / Donut
- Region Exposure: Pie / Donut

### 4. Interaction
- 차트 Hover 시 금액 + 비중 표시
- 클릭 시 상세 화면 이동 없음 (Dashboard는 요약 전용)

### 5. UX Rules
- 모든 금액은 KRW 기준
- 소수점 2자리 고정
